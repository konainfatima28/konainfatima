## 🚀 Flagship Project

### [agentic-pdf-rag-3.0](https://github.com/konainfatima28/agentic-pdf-rag-3.0) — Agentic AI System for Multi-Modal PDF Understanding

> Multi-agent pipeline that deeply understands complex PDFs — tables, charts, embedded images, and mixed layouts — with full reasoning traces and page-level source citations.

**Architecture highlights:**
- Custom **LangGraph-style ReAct loop** with tool-calling agents
- **Multi-agent pipeline**: ingestion agent → vision agent → retrieval agent → answer agent
- **Preprocessing at ingestion time** — table summarization, image captioning, document overview — so query-time stays fast
- **ChromaDB** vector store with `nomic-embed-text` embeddings via Ollama
- **LLaVA vision model** for chart and diagram captioning
- **Page-level source citations** with jump navigation in Streamlit UI
- Full **reasoning trace visibility** per answer

**Stack:** `pdfplumber` · `PyMuPDF` · `ChromaDB` · `Ollama` · `LLaVA` · `Streamlit` · `OpenAI`
